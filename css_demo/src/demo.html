Essential Page Layout

</h3> <h4>

Every page's layout should look like this:

</h4> 
                            
                            <!DOCTYPE html>
                            <html>
                                <head>
                                <stuff>
                                <more stuff...>
                                </head>

                                <body>
                                    <nav>
                                        Nav stuff
                                    </nav>
                                    <div id="page-container">
                                        <div class="content-wrap">
                                            <!-- all other page content -->
                                        </div>

                                        <footer></footer>
                                    </div>
                                </body>

                            </html>
<h4>

Why should it look like that?

</h4>

page-container should wrap everything in the page body that comes after the nav, including the footer.

Nested inside page-container is content-wrap. Basically,  all the content that you add to a webpage will be nested in the content-wrap tags.

This structure was designed so that in all cases, the page creates the optimal height such that the footer stays at the bottom.

Cards are created with rows and columns

Rows and columns

I use a flow container to contain rows. Rows and columns are Bootstrap classes which you can read more about here.

Basically if you want two cards to sit next to each other, you create two columns nested in one row:

Code:

                                
                                <div class="row">
                                    <div class="col">Card 1</div>
                                    <div class="col">Card 2</div>
                                </div>

Result:

Card 1
Card 2


But wait...they aren't really cards yet!

Card-rows and cards

You need to use my special classes card-row and card in conjunction with the Bootstrap classes:

Code:

                                
                                <div class="row card-row">
                                    <div class="col card">Card 1</div>
                                    <div class="col card">Card 2</div>
                                </div>

Result:

Card 1
Card 2


Space them out

Space Between the Cards

The cards have no room for Jesus between them! I add a bit of space between them like this:

Code:

                                
                                <div class="row card-row">
                                    <div class="col card">Card 1</div>
                                    <div class="col-xs"><p>&emsp;&emsp;</p></div>
                                    <div class="col card">Card 2</div>
                                </div>
